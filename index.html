Overview

This is a small collection of single-file jQuery plugins I have extracted from various projects; see below for links to demos and documentation. Any requirements for supporting CSS rules are documented in the Javascript source files themselves.

  • Auto-growing textarea
  • Cookie-based style switcher
  • DOM builder
  • Input hint
  • Rollover
  • Miscellaneous text effects - typewriter, unscramble, ROT13
