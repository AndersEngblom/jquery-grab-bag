Cookie Style Switcher

jQuery based helper function for implementing text resize functionality and storing the user's preference in a cookie. The selected "size" is applied to the document by adding a corresponding CSS class to document.body; see below for example code.

Call the initialisation function (cookieResize) from within your $(document).ready(), passing a jQuery selector identifying the elements the user will click to switch styles and the default class to apply if no cookie is present. Each of the actuator elements should have a single CSS class; this will be applied to document.body and stored in a cookie on click.

Example HTML for size control:
<div id='sizer'>
  <a href='#' class='small'>small</a>
  <a href='#' class='medium'>medium</a>
  <a href='#' class='large'>large</a>
</div>
Example CSS:
body.small { font-size: 10px }
body.medium { font-size: 12px }
body.large { font-size: 14px }
Javascript for initialisation:
$(function() { cookieResize('#sizer a', 'medium'); });
