Input Hint

Display hints to users within a text input and remove them automatically on focus/form submission. This implementation relies on a non-standard attribute, hint; if displaying a VALID XHTML badge is precious to you this is going to mess up your day.

Example using hint attribute:
HTML:
<input type='text' name='forename' hint='Forename' />
          
JS:
$('*[@hint]').inputHint();

Example using auxiliary element:
HTML:
<input type='text' name='forename' />
<kbd>Enter your forename here</kbd>
          
JS:
$('input').inputHint({using: '+kbd'});
Enter your forename here
Enter your surname here
(obviously you'd hide the <kbd/> elements in real usage)
